Use this on the leader Mac from the project checkout.

Terminal 
cd /Users/ruturaj_vasant/Desktop/Academic/DS_Project

In the leader terminal, use workers to list joined workers and start 5 to run five epochs.

Supported Workloads

These values can be used with --distributed-dataset and --distributed-model.

Datasets

cifar10 for quick checks and baseline experiments.

cifar100 for a harder image-classification run.

tiny-imagenet-200 for the main research experiments.

Models

cifar_cnn for smoke tests and local debugging.

resnet50 and resnet101 for heavier data-parallel and pipeline runs.

vit_b_16 for ViT data-parallel and 2-stage pipeline experiments.

For TinyImageNet with ResNet or ViT, use --distributed-image-size 224. Every worker loads the dataset locally under .data/.

Federated Mode

MNIST model averaging with per-run epoch, batch, and learning-rate overrides in the leader terminal.

Federated MNIST

Terminal 
./installations/leader_macos.sh \
  --mode federated \
  --train-batches-per-epoch 200 \
  --batch-size 64 \
  --epochs 5 \
  --eval-batches 40

Runtime command examples: start, start 5, or start epochs=5 batches=100 lr=0.03.

Distributed Data Parallel

Synchronous Gloo all-reduce. Use --distributed-optimizations to switch experiment variants.

No Optimization

Data 
./installations/leader_macos.sh \
  --mode distributed \
  --distributed-parallel data \
  --distributed-dataset cifar10 \
  --distributed-model resnet50 \
  --distributed-image-size 64 \
  --distributed-samples 10000 \
  --distributed-batches-per-epoch 100 \
  --distributed-eval-batches 20 \
  --distributed-optimizations none \
  --dist-master-addr leader-macbook-pro.taila5426e.ts.net \

Pipeline Parallel

Two-stage ResNet50, ResNet101, or ViT-B/16 pipeline with one leader and one worker.

Two-Stage Pipeline

Pipeline 
./installations/leader_macos.sh \
  --mode distributed \
  --distributed-parallel pipeline \
  --distributed-dataset cifar10 \
  --distributed-model resnet50 \
  --distributed-image-size 64 \
  --distributed-samples 10000 \
  --distributed-base-batch 8 \
  --distributed-microbatch-size 2 \
  --distributed-batches-per-epoch 100 \
  --distributed-optimizations none \
  --dist-master-addr leader-macbook-pro.taila5426e.ts.net \
  --epochs 5

Pipeline mode ignores data-parallel optimization modes because it does not all-reduce gradients.

Experiment Matrix

Start one worker first, then run the full matrix. Reports are saved under runs/.

Real Two-Machine Matrix

Experiment 
./experiments/run_experiments.sh

Local Smoke Matrix

Experiment 
./experiments/run_experiments.sh \
  --loopback \
  --model cifar_cnn \
  --image-size 32 \
  --skip-pipeline

Ports: workers need the leader control port 8787. Distributed training also needs --dist-port 29501 reachable between leader and workers.
