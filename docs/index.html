Download a worker bootstrap script.

Pick the script for the machine that will join the distributed training cluster. Each bootstrap sets up dependencies, connects through Tailscale, and starts the worker.

macOS Worker

bootstrap_macos.sh
Download for macOS

Run from Terminal

curl -L -O https://raw.githubusercontent.com/Ruturaj-Vasant/Distributed_MachineLearning_Training_Inference/main/bootstrap_macos.sh
chmod +x bootstrap_macos.sh
./bootstrap_macos.sh

Windows Worker

bootstrap_windows.ps1
Download for Windows

Run from PowerShell

Invoke-WebRequest -Uri "https://raw.githubusercontent.com/Ruturaj-Vasant/Distributed_MachineLearning_Training_Inference/main/bootstrap_windows.ps1" -OutFile "bootstrap_windows.ps1"
powershell -ExecutionPolicy Bypass -File .\bootstrap_windows.ps1

The scripts connect to the default leader configured in the project. Set LEADER_HOST and LEADER_PORT before running a script when testing against a different leader.
