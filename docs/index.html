Recommended Order

The leader can run alone for baselines. For multi-node runs, start the worker first, then start the leader experiment.

1. Leader

Start the leader on the Mac that workers can reach over Tailscale.

2. Worker

Run the worker bootstrap once on each compute machine.

3. Matrix

Run the standard experiments and collect CSV, TXT, and PNG outputs from runs/.
