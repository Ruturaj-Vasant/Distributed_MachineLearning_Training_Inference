Distributed ML Training Leader and worker commands
GitHub repository

Run the cluster.

Start the leader on the Mac first, then run the command block for each worker operating system. Worker setup stages downloads in a temporary folder and keeps the reusable runtime environment for future runs.

Leader

Leader Mac

Terminal 
cd /Users/ruturaj_vasant/Desktop/Academic/DS_Project
source .venv/bin/activate

./leader_macos.sh \
  --host 0.0.0.0 \
  --mode distributed \
  --dist-master-addr leader-macbook-pro.taila5426e.ts.net \
  --dist-port 29501

In the leader terminal, use workers to see joined workers and start 5 to run 5 epochs.

Workers

macOS

Terminal 
set -e
work="$HOME/dml-worker"
runtime="$work/Distributed_MachineLearning_Training_Inference-main"
stage="$(mktemp -d)"
cleanup() { [ -n "$stage" ] && rm -rf "$stage"; }
trap cleanup EXIT
mkdir -p "$runtime" && \
curl -L "https://github.com/Ruturaj-Vasant/Distributed_MachineLearning_Training_Inference/archive/refs/heads/main.zip" -o "$stage/worker.zip" && \
unzip -q -o "$stage/worker.zip" -d "$stage" && \
rsync -a --delete --exclude ".venv/" --exclude ".data/" --exclude ".dml_worker_id" "$stage/Distributed_MachineLearning_Training_Inference-main/" "$runtime/" && \
cd "$runtime" && \
chmod +x bootstrap_macos.sh && \
./bootstrap_macos.sh

Windows

PowerShell 
$work = Join-Path $HOME "dml-worker"
$runtime = Join-Path $work "Distributed_MachineLearning_Training_Inference-main"
$stage = Join-Path ([IO.Path]::GetTempPath()) ("dml-worker-install-" + [guid]::NewGuid().ToString("N"))
New-Item -ItemType Directory -Force -Path $runtime, $stage | Out-Null
try {
  Invoke-WebRequest -Uri "https://github.com/Ruturaj-Vasant/Distributed_MachineLearning_Training_Inference/archive/refs/heads/main.zip" -OutFile (Join-Path $stage "worker.zip")
  Expand-Archive -Path (Join-Path $stage "worker.zip") -DestinationPath $stage -Force
  $source = Join-Path $stage "Distributed_MachineLearning_Training_Inference-main"
  Get-ChildItem -Force $runtime | Where-Object { @(".venv", ".data", ".dml_worker_id") -notcontains $_.Name } | Remove-Item -Recurse -Force
  Get-ChildItem -Force $source | Copy-Item -Destination $runtime -Recurse -Force
  Set-Location $runtime
  powershell -ExecutionPolicy Bypass -File .\bootstrap_windows.ps1
} finally {
  Remove-Item -Recurse -Force $stage -ErrorAction SilentlyContinue
}

Default leader: leader-macbook-pro.taila5426e.ts.net:8787

For a different leader, set LEADER_HOST and LEADER_PORT before running the final bootstrap command.
