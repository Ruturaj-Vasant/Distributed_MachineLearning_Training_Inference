Distributed ML Training Worker network bootstrap
GitHub repository

Join a worker node.

Run the command block for your operating system. It downloads the latest project code, installs the required local dependencies, authenticates Tailscale when needed, and starts the worker process.

macOS

Terminal 
mkdir -p "$HOME/dml-worker" && \
cd "$HOME/dml-worker" && \
curl -L "https://github.com/Ruturaj-Vasant/Distributed_MachineLearning_Training_Inference/archive/refs/heads/main.zip" -o worker.zip && \
unzip -q -o worker.zip && \
cd Distributed_MachineLearning_Training_Inference-main && \
chmod +x bootstrap_macos.sh && \
./bootstrap_macos.sh

Windows

PowerShell 
$work = Join-Path $HOME "dml-worker"
New-Item -ItemType Directory -Force $work | Out-Null
Set-Location $work
Invoke-WebRequest -Uri "https://github.com/Ruturaj-Vasant/Distributed_MachineLearning_Training_Inference/archive/refs/heads/main.zip" -OutFile "worker.zip"
Expand-Archive -Path "worker.zip" -DestinationPath "." -Force
Set-Location ".\Distributed_MachineLearning_Training_Inference-main"
powershell -ExecutionPolicy Bypass -File .\bootstrap_windows.ps1
